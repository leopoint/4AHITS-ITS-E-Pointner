## 1. Übung (VNC - Recherche)
VNC (Virtual Network Computing) ist ein Protokoll, mit dem man einen Computer über ein Netzwerk fernsteuern kann. Dabei sieht man den Bildschirm des anderen Computers und kann Maus und Tastatur benutzen.

**Wichtige Eigenschaften**
- arbeitet mit dem RFB-Protokoll (Remote Frame Buffer)
- grafischer Fernzugriff
- plattformübergreifend (Linux, Windows, macOS)
- Client/Server Architektur

**Komponenten**
- VNC Server → läuft auf dem Zielsystem (Metasploitable)
- VNC Viewer (Client) → verbindet sich zum Server (z. B. von Kali)

**Standardport**
- TCP 5900

## 2. Übung (ssh auf Metasploitable)

public/private key Paar generieren
```sh
ssh-keygen -t rsa -b 2048
```
- -t rsa zwingt RSA
- -b 2048 = starke Länge

Nun muss man eine Datei erstellen, in der das Paar gespeichert wird. 
Namen: "id_rsa" => 
"id_rsa" und "id_rsa.pub" erstellt
 
Da nun beide Files im Home Verzeichnis sind, moved man diese in ein erstelltes Directory 
und legt die Rechte fest.
```sh
$ mkdir ~/.ssh
$ chmod 700 ~/.ssh          
$ mv id_rsa ~/.ssh
$ chmod 600 ~/.ssh/id_rsa
$ mv id_rsa.pub ~/.ssh
$ chmod 644 ~/.ssh/id_rsa.pub
```
```sh
ssh-copy-id -i ~/.ssh/id_rsa msfadmin@192.168.14.51  
```
Es entsteht ein fehler da die neue Kali version zu neu ist und nur über eine alte Funkioniert 
WICHTIG dieser COMMAND funktioniert nur einmal! =>
```sh
ssh-copy-id -o HostKeyAlgorithms=+ssh-rsa -o PubkeyAcceptedAlgorithms=+ssh-rsa -i ~/.ssh/id_rsa.pub msfadmin@192.168.14.51
```

**TESTEN**
```sh
ssh -o HostKeyAlgorithms=+ssh-rsa msfadmin@192.168.14.51
```
