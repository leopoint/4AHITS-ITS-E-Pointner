 ## Github verlinken und zugreifen
 1. ```sh
        sudo apt update
    ```
- Aktualisiert die Paketlisten

2. ```sh
        sudo apt install -y git
    ```
- installiert git

3. ```sh
        git config --global user.name "Leo.Pointner"
        git config --global user.email "leopointner2007@gmail.com"
    ```
- Bei GITHUB anmelden

4. ```sh
        git clone https://github.com/leopoint/4AHITS-ITS-E-Pointner.git
    ```
- mit git clone Github auf Kali clonen und zum bearbeiten bereitstellen
- Repository Pfad angeben
